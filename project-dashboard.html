課程後台 Dashboard UI 設計

模擬線上學習平台的課程與學員管理介面

專案類型

後台 Dashboard UI

負責角色

介面規劃・前端切版

使用技術

HTML / CSS / JavaScript

專案目的

資訊結構與操作流程設計

專案背景與需求

此專案模擬線上學習平台的後台管理系統， 包含課程管理、學員列表與基本數據呈現。

設計重點不在視覺裝飾， 而是讓管理者能快速理解狀態、 有效完成日常操作。

解決方案與設計策略

  1. 以資訊層級為優先，避免過度裝飾
  2. 常用操作集中於視覺焦點區域
  3. 表格、卡片與狀態標示清楚分區
  4. 考量日常使用情境，維持介面穩定性

成果與學習

透過此專案，我練習了 Dashboard 常見版型結構， 並理解後台介面在資訊呈現與操作效率上的設計重點。

也加深了對 Bootstrap Grid、 元件模組化與實務管理介面設計的理解。
